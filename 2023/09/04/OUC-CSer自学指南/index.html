OUC_CSer自学指南

发表于 2023-09-04 阅读次数：

前言

: ) 一年前曾经在pyq说过：“等我变强了要写一个OUC CSer自学指南”，然而一年过去了实际上我也没有变强多少，只是会一些CRUD罢了。写这个的主要目的是想帮助后面的同学更好地认识CS，不至于说讨厌（因为我知道不少同学来了这个专业已经打算转行了）。再有一个目的是，帮助大家扎实基础，侯捷老师说过“勿在浮沙筑高台”，祝大家在CS的道路上越走越好。

学习路线

注：本路线不涉及公共基础课

大一

广度认知

刚入学学的是《计算机科学导论》，其实现在想想，确实是一门不错的课，但一些课程作业真的有点太难了。我这里推荐学习《Crash Course Computer Science》，B站有视频。https://www.bilibili.com/video/BV1EW411u7th?vd_source=99690035e06ef0ba562b557f372ceec0。主要是从各个角度理解计算机，教给学生一个CS MAP，虽然讲的不深，但是一定程度上扩产了一个小白的计算机知识。

《MIT-Missing-Semester》：教你一些计算机常用的工具，git、shell等 https://www.bilibili.com/video/BV1x7411H7wa?vd_source=99690035e06ef0ba562b557f372ceec0

编程语言

学习方法：学完一门编程语言，最重要的是要用起来，计算机的出现是为了解决需求，所以学完了可以去GitHub找一个小项目进阶。

C语言

让一个刚入门CS的学生就上手C语言，实在感觉有点不合适。在四大(MIT, CMU, UCB, Stanford)的入门课程中，一般都是使用python入门。

我在这里推荐下UCB的CS61A https://cs61a.org/

也可以看浙大的C语言课程 https://www.bilibili.com/video/BV1dr4y1n7vA?vd_source=99690035e06ef0ba562b557f372ceec0

C++

我推荐的C++课程是侯捷老师的C++！Youtube有他的课！他的主要路线是C++面向对象—> C++ STL —> C++新标准 C++11 —> C++内存管理 —> C++设计模式

有条件的同学可以去看一下，真的讲的不错。

C++书籍：《C++ primer》 配合B站的课程看。

离散数学

我个人觉得这门课很硬，对未来的课程理解很有帮助。

建议看北大的课程 https://www.bilibili.com/video/BV1BW411n7gw?vd_source=99690035e06ef0ba562b557f372ceec0

还有一位太原理工大学的老师讲的也非常不错 https://www.bilibili.com/video/BV1Z5411h7gb?vd_source=99690035e06ef0ba562b557f372ceec0

大二

计算机系统基础

我认为这门课是《计算机组成原理》和《操作系统》的先修课。通过这门课可以更好的了解计算机一些底层的原理，例如位运算、二进制等。（而且学好了会觉得计组不难

在这里，我重磅推荐CMU的15-213

B站：https://www.bilibili.com/video/BV1iW411d7hd?vd_source=99690035e06ef0ba562b557f372ceec0

课程网站：https://www.cs.cmu.edu/afs/cs/academic/class/15213-f15/www/schedule.html

这门课可能是CMU最火的课了。配合着书籍《深入理解计算机系统》一起学习，然后做lab，会让你的个人能力得到很大的提升。

数据结构

这门课我是通过王道考研课入门【王道计算机考研 数据结构】https://www.bilibili.com/video/BV1b7411N798?vd_source=99690035e06ef0ba562b557f372ceec0，学习理论的同时可以去多写写算法题，leetcode、牛客都是很好的做题平台，当时大二下刷了大概200多道题吧，期末考试的题都是小菜。

另外还推荐Acwing这个平台学习算法，主要是打算法竞赛，像ACM这种。https://www.acwing.com

计算机网络

虽然说在我们的培养方案里面这门课是大三的课，但我个人认为大二可以开始学了。

当时入门是学的中科大的课程 https://www.bilibili.com/video/BV1JV411t7ow?vd_source=99690035e06ef0ba562b557f372ceec0 老师的配套教材也是著名的《计算机网络自顶向下》，醍醐灌顶。

后面大二下我不满足于只学习理论，决定找一些lab做，于是发现了CS144https://cs144.github.io/，这是用C++实现一个TCP协议，视频我没怎么看，看实验指导书做实验就ok，**做完lab可以写简历上，不管是求职还是考研复试都会很加分**。。

操作系统

我个人很喜欢操作系统这门课，能推荐的优质网课资源真是太多了。

大二下的时候通过王道考研的视频入门的理论知识https://www.bilibili.com/video/BV1YE411D7nH?vd_source=99690035e06ef0ba562b557f372ceec0，

RISC-V操作系统：【[完结] 循序渐进，学习开发一个RISC-V上的操作系统 - 汪辰 - 2021春】https://www.bilibili.com/video/BV1Q5411w7z5?vd_source=99690035e06ef0ba562b557f372ceec0

南京大学jyy的课程：【操作系统概述 (操作系统的历史；学习建议) [南京大学2023操作系统-P1] (蒋炎岩)】https://www.bilibili.com/video/BV1Xx4y1V7JZ?vd_source=99690035e06ef0ba562b557f372ceec0

重磅推荐mit6.S081: 当时大二暑假接触到的，通过lab实现一个xv6操作系统https://pdos.csail.mit.edu/6.S081/2020/ ，这门课网上很多博客讲解，教授是有名的Robert Morris，蠕虫病毒出自他手。做完lab可以写简历上，不管是求职还是考研复试都会很加分。

大三

提前学完操作系统、计网，大三就会轻松很多。

数据库

我们学校的数据库主要是教写sql语句，我个人比较喜欢偏工程。

理论的话，推荐学习哈工大战神的课程：https://www.bilibili.com/video/BV1HY4y1b72A?vd_source=99690035e06ef0ba562b557f372ceec0

推荐CMU15445，这门课使用C++实现一个数据库 https://15445.courses.cs.cmu.edu/fall2023/

做完lab可以写简历上，不管是求职还是考研复试都会很加分

分布式系统

这门课我们学校没有开，但是我最近发现早在入学的学科介绍PPT就有提到这门课(三年过去了才发现SOS。这就是大名鼎鼎的mit6.824，go语言实现一个小型分布式。课程非常硬核，主要是读论文并实现。https://pdos.csail.mit.edu/6.824/

做完lab可以写简历上，不管是求职还是考研复试都会很加分！！

后续推荐

emmm以上操作系统、计网、数据库学完就已经是一个不错的CS学生了，你可以通过代码实现一些自己喜欢的东西，不论是工具也好还是一些中间件。 如果你志在就业，你可以学习一些工作会用到的：框架、MySQL、Redis等，多去实习。如果你志在科研，就进实验室看看自己感兴趣的研究方向。

计算机是一个很偏实践的专业，就像游泳和开车一样，不是多看别人就会了，不是说看多少视频就会变强，重要的是要多写代码多做项目，这样才会进步。祝大家在CS的道路上愉快学习:D
